# Trade Compass

Платформа для анализа акций российских компаний с AI помощником

## Core Features

- Анализ секторов с помощью AI
- Анализ компаний на основе большого контекста о ней по фреймворку MorningStar
- Чат с нейроассистентом по отчетам компаний

Платформа располагается по адресу trade-compass.ru. В данное время еще в разработке.
