# Инструкция по фундаментальному анализу акций MOEX

## Обзор методологии

Данная инструкция основана на методологии Morningstar Equity Research и адаптирована для российского фондового рынка (Московская биржа). Цель — определить справедливую стоимость акции и присвоить рейтинг на основе соотношения текущей цены и intrinsic value с учётом уровня неопределённости.

**Четыре ключевых компонента оценки:**

1. Экономический ров (Economic Moat)
2. Справедливая стоимость (Fair Value)
3. Неопределённость (Uncertainty)
4. Текущая рыночная цена

---

## 1. Экономический ров (Economic Moat)

Экономический ров — структурное конкурентное преимущество, позволяющее компании генерировать доходность на вложенный капитал (ROIC) выше стоимости капитала (WACC) в течение длительного периода.

### Источники экономического рва

| Источник | Описание | Примеры на MOEX |
|----------|----------|-----------------|
| **Нематериальные активы** | Бренды, патенты, лицензии, регуляторные барьеры входа | Сбер (банковская лицензия), Полюс (лицензии на месторождения), Мосбиржа (инфраструктурная монополия) |
| **Издержки переключения** | Клиентам дорого или сложно перейти к конкурентам | 1С-экосистема, банковские зарплатные проекты, ERP-системы |
| **Сетевой эффект** | Ценность продукта/сервиса растёт с увеличением числа пользователей | Ozon, Wildberries (маркетплейсы), Яндекс (поиск + такси + еда) |
| **Ценовое преимущество** | Структурно более низкие издержки производства | Норникель (уникальные месторождения), Русал (дешёвая электроэнергия ГЭС), Сибур (доступ к дешёвому сырью) |
| **Эффективный масштаб** | Рынок ограничен, новым игрокам экономически невыгодно входить | Мосбиржа, Транснефть, морские порты, аэропорты |
| **Господдержка** *(РФ-специфика)* | Системная значимость, госзаказы, субсидии, льготное финансирование | Системообразующие банки, Ростех, ОАК, оборонные предприятия |

### Рейтинг экономического рва

| Рейтинг | Критерии |
|---------|----------|
| **Wide (Широкий)** | Высокая уверенность в сохранении excess returns минимум 10 лет, вероятно 20+ лет. Преимущество устойчиво даже при санкционном давлении. |
| **Narrow (Узкий)** | Excess returns вероятны минимум 5-10 лет, но преимущество уязвимо к регуляторным или санкционным шокам. |
| **None (Отсутствует)** | Конкурентное положение неустойчиво, ROIC стремится к WACC в краткосрочной перспективе. |

### Оценка угроз разрушения стоимости

При анализе рва необходимо оценить риски, способные уничтожить экономическую прибыль компании:

- Санкционные риски (SDN, секторальные ограничения)
- Отраслевая дизрупция (технологические изменения)
- Регуляторное давление (windfall taxes, ценовое регулирование)
- Финансовое здоровье (чрезмерный леверидж)
- Идиосинкратические риски (ESG, корпоративное управление)

Если вероятность реализации такого риска высока — компания не может считаться обладающей экономическим рвом.

## 2. Справедливая стоимость (Fair Value)

### Выбор источника отчётности

| Тип отчётности | Когда использовать | Особенности |
|----------------|-------------------|-------------|
| **МСФО (IFRS)** | Основной источник для DCF-анализа | Консолидированная отчётность, справедливая стоимость активов, полная картина бизнеса |
| **РСБУ (RAS)** | Дополнительно для анализа дивидендов | Дивидендная база часто привязана к прибыли по РСБУ, может существенно отличаться от МСФО |

### DCF-модель: три стадии

#### Стадия I — Явный прогноз (5 лет)

Полное моделирование финансовой отчётности:

- Прогноз выручки с учётом отраслевой динамики
- Операционная маржинальность (EBITDA margin, EBIT margin)
- Эффективная налоговая ставка
- Капитальные затраты (Capex)
- Изменения оборотного капитала (ΔWC)
- Амортизация

**Расчёт свободного денежного потока:**

```
FCF = EBIT × (1 - Tax Rate) + D&A - Capex - ΔWorking Capital
```

**Для экспортёров дополнительно моделировать:**

- Прогноз цен на commodities (нефть, газ, металлы, удобрения)
- Прогноз курса USD/RUB (или EUR/RUB, CNY/RUB)
- Сценарный анализ комбинаций цена × курс

#### Стадия II — Затухание (Fade Period)

Период снижения ROIC до уровня WACC. Длительность зависит от экономического рва:

| Рейтинг рва | Длительность Stage II |
|-------------|----------------------|
| None | 1-2 года |
| Narrow | 3-7 лет |
| Wide | 7-12 лет |

На этой стадии используются упрощённые допущения:

- Средний темп роста EBI
- Нормализованная ставка реинвестирования
- Постепенное снижение RONIC до WACC

#### Стадия III — Терминальная стоимость (Perpetuity)

Расчёт продлённой стоимости при ROIC = WACC:

```
Terminal Value = FCF_terminal × (1 + g) / (WACC - g)
```

**Терминальный рост (g) для российских компаний:** 4-6% (привязка к долгосрочному номинальному росту ВВП РФ или целевой инфляции ЦБ + реальный рост)

### Расчёт WACC для российских компаний

```
WACC = (E / (E + D)) × Re + (D / (E + D)) × Rd × (1 - t)
```

**Стоимость собственного капитала (Re):**

```
Re = Rf + β × ERP + CRP
```

| Компонент | Значение | Источник/комментарий |
|-----------|----------|---------------------|
| **Rf** (безрисковая ставка) | 11-14% | Доходность ОФЗ 10 лет |
| **ERP** (премия за риск акций) | 5-6% | Историческая премия для развивающихся рынков |
| **CRP** (страновая премия РФ) | 3-8% | Зависит от санкционного режима и геополитики |
| **β** (бета) | 0.8-1.5 | Отраслевая бета, скорректированная на леверидж |

**Стоимость заёмного капитала (Rd):**

- Для компаний с рейтингом: доходность облигаций компании
- Без рейтинга: ключевая ставка ЦБ + кредитный спред (2-5%)

### Мультипликаторный кросс-чек

DCF-оценку необходимо проверять через сравнительный анализ:

| Мультипликатор | Применимость | Исторический диапазон MOEX |
|----------------|--------------|---------------------------|
| **P/E** | Прибыльные компании со стабильной прибылью | 4-8x (дисконт к EM) |
| **EV/EBITDA** | Капиталоёмкие компании, компании с долгом | 3-6x |
| **P/BV** | Банки, финансовые холдинги | 0.5-1.2x |
| **EV/Reserves** | Нефтегаз, добывающие компании | Секторально специфичен |
| **Div Yield** | Дивидендные истории | 8-15% = привлекательно |
| **P/S** | Растущие компании с отрицательной прибылью | 0.5-2x |

## 3. Неопределённость (Uncertainty Rating)

Рейтинг неопределённости отражает диапазон потенциальных исходов для intrinsic value компании и определяет требуемую маржу безопасности.

### Факторы неопределённости для MOEX

| Категория | Факторы риска |
|-----------|---------------|
| **Санкционные** | SDN-лист, секторальные санкции, вторичные санкции, риск делистинга ГДР, заморозка активов |
| **Регуляторные** | Windfall taxes, экспортные пошлины, демпферы, ценовое регулирование, принудительные дивиденды госкомпаний |
| **Валютные** | Волатильность RUB, ограничения на движение капитала, репатриация выручки |
| **Корпоративные** | Качество раскрытия информации, конфликт интересов миноритариев, риск размытия через допэмиссию |
| **Операционные** | Цикличность бизнеса, commodity-зависимость, импортозависимость, технологические риски |
| **Ликвидность** | Free-float, объёмы торгов, доступ нерезидентов, риск делистинга |

### Шкала неопределённости

| Рейтинг | Описание | Типичные представители |
|---------|----------|----------------------|
| **Low** | Стабильный предсказуемый бизнес, минимальные санкционные риски, высокое качество раскрытия | МТС, Мосбиржа, X5 Retail, Магнит |
| **Medium** | Умеренная цикличность или ограниченные санкционные риски, приемлемая прозрачность | Сбер, Яндекс, девелоперы (ПИК, Самолёт), Ozon |
| **High** | Значительная commodity-зависимость, секторальные санкции, операционный леверидж | Лукойл, НЛМК, Северсталь, Фосагро, АЛРОСА |
| **Very High** | SDN или высокий риск попадания, низкая прозрачность, структурные проблемы | ВТБ, Русал, En+, компании с высоким госучастием |
| **Extreme** | Под жёсткими санкциями, экзистенциальные риски, возможность национализации или банкротства | Газпром, Аэрофлот, оборонные предприятия |

---

## 4. Маржа безопасности и звёздный рейтинг

### Пороговые значения для MOEX

Требуемый дисконт к Fair Value для присвоения рейтинга ★★★★★ (покупка) и премия для ★ (продажа):

| Uncertainty | Дисконт для ★★★★★ | Премия для ★ |
|-------------|-------------------|--------------|
| Low | 25% | 30% |
| Medium | 35% | 45% |
| High | 50% | 70% |
| Very High | 60% | 100% |
| Extreme | 80% | 200% |

*Примечание: пороги выше стандартных Morningstar из-за повышенных страновых рисков РФ.*

### Интерпретация звёздного рейтинга

| Рейтинг | Интерпретация |
|---------|---------------|
| ★★★★★ | Цена значительно ниже справедливой стоимости с учётом риска. Сценарный анализ указывает на чрезмерно пессимистичные ожидания рынка. Высокий потенциал роста при ограниченном downside. |
| ★★★★ | Умеренная недооценка. Доходность выше справедливой risk-adjusted return вероятна. |
| ★★★ | Справедливая оценка. Ожидаемая доходность примерно соответствует стоимости собственного капитала (cost of equity). |
| ★★ | Умеренная переоценка. Доходность ниже справедливой risk-adjusted return вероятна. |
| ★ | Значительная переоценка. Рынок закладывает чрезмерно оптимистичные ожидания. Высокий риск потери капитала. |

## 5. Дивидендный анализ

Дивидендная доходность — ключевой фактор привлекательности акций на MOEX для розничных инвесторов.

### Дивидендный чек-лист

| Параметр | Что анализировать |
|----------|-------------------|
| **Дивидендная политика** | Формула расчёта (% от ЧП МСФО / РСБУ / FCF / EBITDA), периодичность выплат, история соблюдения политики |
| **Payout ratio** | Доля прибыли, направляемая на дивиденды. 40-60% — комфортно, >80% — риск сокращения |
| **Дивидендная база** | МСФО vs РСБУ — определить, что выгоднее акционерам в текущем периоде |
| **Денежное покрытие** | FCF / Дивиденды > 1.0 — дивиденды обеспечены реальным денежным потоком |
| **Долговая нагрузка** | Net Debt / EBITDA — при высоком леверидже дивиденды под угрозой |
| **Госвлияние** | Минфин требует 50% от ЧП МСФО от госкомпаний (не всегда выполняется) |
| **Санкционные ограничения** | Возможность физически выплатить дивиденды (пример: Газпром 2022) |

### Интерпретация дивидендной доходности

| Div Yield | Интерпретация |
|-----------|---------------|
| < 6% | Низкая для российского рынка — требуется убедительный growth-кейс |
| 6-10% | Норма для качественных дивидендных историй |
| 10-15% | Привлекательно, но необходимо проверить устойчивость выплат |
| > 15% | Рынок, вероятно, закладывает сокращение дивидендов — требуется глубокий анализ причин |

## 6. Качество управления капиталом (Capital Allocation)

### Критерии оценки

| Аспект | Позитивные сигналы | Негативные сигналы |
|--------|-------------------|-------------------|
| **Баланс** | Net Debt / EBITDA < 2x, валютный долг захеджирован, комфортный график погашения | Высокий леверидж, FX-mismatch (валютный долг при рублёвой выручке), короткий долг |
| **Инвестиции** | Прозрачная инвестпрограмма, разумные M&A с чёткой логикой, дисциплина в оценке | "Стройки века", покупки по завышенным ценам, сделки со связанными сторонами |
| **Возврат акционерам** | Стабильные дивиденды, байбэки ниже справедливой стоимости | Невыплата дивидендов при наличии кэша, размытие через допэмиссии, байбэки на пике |
| **Governance** | Независимые директора, прозрачная отчётность, учёт интересов миноритариев | Related party transactions, конфликты с миноритариями, непрозрачные схемы |

### Рейтинг Capital Allocation

| Рейтинг | Описание | Примеры |
|---------|----------|---------|
| **Exemplary** | Образцовое отношение к миноритариям, дисциплинированное распределение капитала | Лукойл, Новатэк, Сбер |
| **Standard** | Приемлемое управление без явных нарушений интересов миноритариев | Большинство публичных компаний |
| **Poor** | История ущемления миноритариев, сомнительные сделки, непрозрачность | Исторически: Россети, ряд госкомпаний |

## 7. Санкционный профиль

### Санкционный скоринг

| Уровень | Описание | Влияние на оценку |
|---------|----------|-------------------|
| **0 — Чисто** | Нет санкций, минимальный риск попадания, внутренний бизнес | Без корректировок |
| **1 — Риск** | Сектор под давлением, компания пока не затронута | WACC +1-2%, мониторинг |
| **2 — Секторальные** | Ограничения на долгосрочное финансирование, технологии, отдельные товары | WACC +2-3%, дисконт к мультипликаторам 10-20% |
| **3 — SDN частичный** | В санкционном списке, но операционно функционирует | WACC +4-5%, Uncertainty ≥ Very High |
| **4 — SDN жёсткий** | Критические ограничения на операционную деятельность | Uncertainty = Extreme, отдельная модель оценки |

### Факторы потенциального смягчения (Upside-сценарии)

При анализе учитывать возможные позитивные изменения:

- Вероятность и сроки смягчения санкций
- Успешность переориентации экспорта (Европа → Азия)
- Прогресс импортозамещения критических компонентов
- Адаптация логистических цепочек
