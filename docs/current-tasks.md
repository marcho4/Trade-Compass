## Чат на плаформе

1. Обеспечить защиту от бесконечных регистраций с помощью подтверждения по e-mail и сохранению каких-нибудь IP/других метрик при регистрации, чтобы меня не могли абузить.
2. Найти готовый компонент для поддержки чатов с AI.
3. Реализовать API для создания чатов и отсылки запросов с учетом контекста чата.
4. Реализовать рейт лимит на это API и отображать Usage в кабинете.
5. Сделать API доступным только с валидным JWT токеном.
6. Если запрос к AI провайдеру не проходит - usage не списывать.
7. В зависимости от страницы, на которой открыт чат, передавать в контекст доп данные с платформы.
8. Создать MCP для всего внутреннего API и подключить его к чату.
9. Подключить метрики для AI использования, чтобы смотреть сколько использовано токенов.

## Сценарные анализы с рассчетом стоимости исходя из предполагаемых будущих метрик

1. Вытягиваем потенциальные факторы роста
2. Вытягиваем сценарии из новостей сектора
3. Вытягиваем сценарии из новостей зависимостей
4. Вытягиваем сценарии из новостей компании
5. Для каждого сценария берем текущие метрики и говорим нейросети составить прогноз на ближайшие пару лет с учетом сценария. Затем по этим метрикам считается DCF оценка.
