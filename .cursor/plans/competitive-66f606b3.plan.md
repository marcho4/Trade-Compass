# План преодоления конкурентов

## 1. Рыночное позиционирование

- Сфокусироваться на продвинутых инвесторах РФ, которым нужны глубина данных и AI-аналитика по разумной цене (ниже Fiscal.ai, выше Finam для премиума).
- Предлагать freemium: бесплатный анализ 3 компаний + лимитированный портфель; платный «Pro» (2 999 ₽/мес) с безлимитным анализом, алертами и ребалансировкой.

## 2. Дифференциирующие фичи

1. Полное и актуальное MOEX-покрытие (10-летняя история метрик, облигации, ETF)
2. AI-чат внутри каждой страницы компании с контекстом отчётов + «что-если» сценарии
3. Интеллектуальная ребалансировка портфеля (алгоритм на основе целей риска/доходности)
4. Кастомные графики любых фундаментальных метрик (как Fiscal.ai) + сохранение пресетов
5. Сравнение с отраслью «в один клик» (heat-map + quant-rating)
6. Smart-Alerts: события отчётов, пробой уровней, изменение оценки snowflake
7. Интеграции с брокерами (Alfa, Тинькофф, Финам) для автосинхронизации сделок

## 3. Техническая дорожная карта

- Модуль `data-ingestion`: краулер + API MOEX, кэш 1 мин, 10-летние исторические метрики
- Микросервис `ai-analyzer` (Go) обёртка над Claude 3.5 с кэшем промптов
- Сервис `portfolio-core` (Python, CQRS) с ребалансировкой и алерт-движком
- Frontend: Next 14 + shadcn/ui; фокус на mobile-first и p95 <2 s
- Billing: Stripe RU / YooKassa, тарифы и trial-7-days
- Observability: Prometheus + Grafana, лог стоимости AI

## 4. Go-to-Market

- Запуск public beta за 0 ₽ на 1 мес для early adopters
- Контент-маркетинг: кейсы «как обыграть индекс» (+SEO «анализ акций MOEX»)
- Партнёрства с блогерами и брокерами (white-label виджеты)

## 5. Метрики успеха

- DAU ≥ 5 000 через 6 мес
- Конверсия free→Pro ≥ 7 %
- AI-затраты ≤ 20 ₽/анализ
- Отказоустойчивость API > 99.9 %
